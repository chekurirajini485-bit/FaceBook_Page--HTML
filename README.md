## Facebook Login & Account Pages (HTML Only)
## Project Overview
This project is a static Facebook-style authentication interface created using only HTML.
It includes multiple pages that simulate a basic user authentication flow such as login, account creation, and password recovery.
The purpose of this project is to practice HTML form elements, page navigation, and layout structuring without using CSS frameworks or JavaScript.
## Pages Included
The project consists of the following HTML pages:
##Login Page
Email or Phone input
Password input
Login button
Links to Forgotten Password and Create New Account
##Create New Account Page
First Name and Last Name
Mobile number or Email
Password field
Date of Birth selection (Day, Month, Year)
Gender selection (Male, Female, Others)
Sign Up button
Back to Login link
##Forgot Password Page
Email or Phone input
Search button
Back to Login link
